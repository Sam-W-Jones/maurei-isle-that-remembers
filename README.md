# Maurei, The Isle That Remembers — Foundry VTT Module

**Target:** Foundry VTT V12 · D&D5e 3.3.x

## Installation

1. Extract the folder `maurei-isle-that-remembers` into Foundry's `Data/modules/` directory.
2. Enable **Maurei, The Isle That Remembers** in a D&D5e world.
3. Log in as GM. The importer will prompt automatically.
4. Click **Import Adventure**.

The importer creates world folders containing:

- 11 Scenes, including GM and player versions of the island overview and Pale Court
- GM Guide, Player Primer, and Monster Reference journals
- Saltwyrm, Mistling, Hollow, Pale Sentinel, Huntsman, Court's Sword, Hunter Shark, Hauiri, and the sailing ship
- A Mist-Events roll table
- Fading Tracker, Fading Save, Reef Clock, Mist-Event, and Refresh Import macros
- GM map pins and hidden encounter tokens

## Re-importing

Use **Game Settings → Import / Refresh Maurei** or the **Maurei — Refresh Import** macro. Re-importing deletes and recreates only content marked as generated by this module; edits to generated content will be replaced.

## Notes

- The encounter maps use conservative default grids because the generated artwork does not have a mathematically fixed grid. Adjust Scene Configuration → Grid if desired.
- Hauiri is intentionally represented as an Actor for placement and reference, but is immune to all damage and is not a combat encounter.
- Token PNGs have true alpha backgrounds. The grouped spectral sheet has been separated into individual token files under `assets/tokens/echoes/`.
